# Linked List

## Tech Stack

- React, Vite, TypeScript
- Tailwind CSS, Radix UI
- Supabase (PostgreSQL, Realtime, Edge Functions)
- Resend
- Vitest, Playwright

## Setup

```bash
git clone https://github.com/harryden/linked-list.git
cd linked-list
npm install
cp .env.example .env
npm run dev
```

Port: `8080`
